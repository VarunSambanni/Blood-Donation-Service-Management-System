# Blood-Donation-Service-Management-System

A system designed to facilitate the distribution of blood. Organisers add blood donation events, Donors register for events and perform donations, Admin signs up new organisations, performs receptions and view the required tables. Developed using React, NodeJS, ExpressJS, MySQL and MongoDB.
